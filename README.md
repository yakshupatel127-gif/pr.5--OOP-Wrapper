# pr.5--OOP-Wrapper
## Project Description

OOP Wrapper is a simple **Employee Management System** created to understand basic **Object-Oriented Programming (OOP)** concepts.

## Objective

* Learn how OOP works in real applications
* Use classes, inheritance, and encapsulation
* Manage different types of employees

## Classes Used

### Employee

* Stores employee details like ID, name, age, and salary
* Uses getters and setters for secure data handling

### Manager

* Inherits from Employee
* Adds department information

### Developer

* Inherits from Employee
* Adds programming language information

## OOP Concepts Covered

* Classes and Objects
* Inheritance
* Encapsulation
* Method Overloading
* Method Overriding

## Purpose

This project is made for **learning and practice** of OOP concepts for students.
